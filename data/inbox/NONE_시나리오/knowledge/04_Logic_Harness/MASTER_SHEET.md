| **이름의 기원** | 어머니가 사랑한 남자(한국인)가 좋아하던 가수 **조덕배**의 성(Jo)을 딴 것 |

| **도덕적 결함 (Moral Flaw)** | 가족(혈연)보다 조직의 룰을 우선시하는 기계적 비겁함 |
| **청각적 낙인** | 조덕배의 '꿈에' — 어머니의 그리움이 조의 정체성에 새겨진 노래 |

## 2. 니코 (NIKO) — [B안 확정] ✅ [사령관 확정]

| 항목 | 확정 내용 |
|------|----------|
| **정체** | 고아 출신 코피노. 혼혈 고아들을 지키기 위해 빈민가 패거리를 조직한 리더 |
| **조와의 관계** | **가츠(조) ↔ 그리피스(니코)** 구조. 조는 제1 행동대장 |
| **조직의 성격** | 동남아로 진출한 **한국 거대 조폭 세력** 밑의 용병 조직 |
| **B안 핵심** | 니코는 한국 조폭을 **증오**하며 그 세력을 무너뜨리고 동남아를 직접 장악하려는 욕망을 품음 |
| **굴종의 아이러니** | 장악하고 싶은 조직 앞에서 기꺼이 고개를 숙이고 '개'로 복무하는 이중성 |
| **몰락 트리거** | 조가 블록 05에서 최준혁을 처단 → 최준혁 아버지 → 박왕렬 → 한국 조폭에게 명령 → 니코 조직 괴멸 |
| **몰락 방식** | 제1 행동대장(조) 부재 중 한국 조폭 선제 공격 → 니코 생포/조직 괴멸 |
| **배신 (이클립스)** | 한국 조폭 제안: "조와 아이들을 넘겨라, 동남아 마약왕 자리를 주겠다" → 니코: **"바친다."** |
| **배신의 핵심** | 니코가 반역한 것이 아니다. 조의 임무 수행이 형제를 죽인 것이다. 가장 처참한 타락 |

```
엘런 의뢰 (공식 선수금 입금)
    ↓
조 — [블록 05] 최준혁 처단 (타이어 화형)
    ↓
최준혁 아버지 분노 → 조의 정체 파악
    ↓
박왕렬에게 압력
    ↓
박왕렬 → 동남아 한국 조폭에게 명령:
"필리핀 용병단(니코 조직) 괴멸시켜라"
    ↓
조가 한국 원정 중인 동안
필리핀 본진 — 제1 행동대장 부재 → 무방비
    ↓
한국 조폭 선제 공격 → 니코 조직 괴멸 / 니코 생포
    ↓
한국 조폭 제안: "조와 아이들을 넘겨라"
    ↓
니코: "바친다." — 현대판 이클립스
    ↓
조 각성 → 복수의 화신
```
