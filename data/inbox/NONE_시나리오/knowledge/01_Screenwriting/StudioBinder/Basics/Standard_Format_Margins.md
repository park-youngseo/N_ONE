# 시나리오 표준 포맷 (Standard Format & Margins)

시나리오는 업계 표준 규격을 엄격히 준수해야 합니다. 이는 1페이지가 실제 촬영 시간 약 1분에 해당하도록 설계되었기 때문입니다.

## 1. 폰트 및 기본 설정
*   **Font**: 반드시 **Courier 12pt**를 사용해야 합니다. (모든 글자의 너비가 동일한 고정 폭 폰트)
*   **Paper Size**: US Letter (8.5" x 11") 기준.
*   **Spacing**: 줄 간격은 Single Space를 기본으로 합니다.

## 2. 표준 여백 규격 (Industry Standard)
| 요소 | 왼쪽 여백 | 오른쪽 여백 |
| :--- | :--- | :--- |
| **Scene Heading / Action** | 1.5 인치 | 1.0 인치 |
| **Character Name** | 3.7 인치 | - |
| **Dialogue** | 2.5 인치 | 2.5 인치 |
| **Parentheticals** | 3.1 인치 | 3.1 인치 |

## 3. 작성 예시 (Format Demo)
```text
INT. WRITER'S ROOM - NIGHT

A messy desk. A lone laptop glows in the dark. 
The cursor BLINKS.

                        GRAVITY
                (typing)
        Let's build the perfect script.
```

## 핵심 요약 (StudioBinder Basics)
| 원칙 | 상세 내용 |
| :--- | :--- |
| **Font Consistency** | Courier 12pt는 전 세계 공통 약속입니다. |
| **White Space** | 여백이 많을수록 가독성이 좋아지며 촬영 시간이 정확해집니다. |
