# 명작의 테마적 일관성 분석 (Masterpiece Thematic Analysis)

명작 시나리오들이 어떻게 하나의 '중심 생각'을 시각적 서사로 치환하는지 깊이 있게 분석합니다.

## 1. 기생충 (Parasite) - 수직적 알레고리
*   **The Line (선)**: 가난한 자들이 넘지 말아야 할 선을 '냄새'라는 감각적 매개체로 시각화합니다. (Category: Theme)
*   **Vertical Allegory**: 반지하, 저택, 지하실이라는 물리적 공간의 높낮이를 통해 계급 구조를 극명하게 보여줍니다. (Category: Structure)
*   **Parasitic Dynamic**: 누가 누구에게 기생하는가? 부유한 박 사장 가족 역시 가난한 자들의 노동 없이는 일상을 유지할 수 없는 공생적 기생 관계임을 폭로합니다. (Category: Theme)

## 2. 다크 나이트 (The Dark Knight) - 질서 vs 혼돈
*   **Personification of Chaos**: 조커는 혼돈 그 자체이며, 배트맨이 구축한 질서와 도덕적 가치관을 끊임없이 시험합니다. (Category: Character)
*   **The Breaking Point**: 인간의 도덕성이 한계 상황에서도 유지될 수 있는가? 하비 덴트의 타락을 통해 질서의 취약성을 증명합니다. (Category: Theme)

| 작품 | 중심 테마 | 분석 포인트 |
| :--- | :--- | :--- |
| **Parasite** | Class Struggle | 공간과 감각(냄새)을 통한 계급의 시각화 |
| **The Dark Knight** | Chaos vs Order | 캐릭터의 신념을 파괴하는 도덕적 딜레마 설계 |
