궁금하다 하시는 분들은이 화살표들을
눌러서 확인을 하시면 되고요. 왼쪽에
해당 코드들이 만들어지고 있는 걸
확인할 수가 있습니다. 그래서이
프롬프트가 끝날 때까지 여러분들이
읽어 보시고 안티그래비티가 결과물을
만들 수 있게 어프루브를 계속 해
주시면 됩니다. 자, 그리고 안티
그래비티를 사용해서 만드시다 보면
가끔씩 이렇게 이미지를 만드는데
실패했다라는 메시지를 받으실 수도
있을 겁니다. 요것도 제가 웹사이트가
다 만들어지고 해결 방법을 바로 보여
드리겠습니다. 자, 그리고 마지막에
내가 npm런 빌드를 해서 테스트를
하겠다라고 하시면 리젝트를 눌러
주시면 됩니다. 자, 그러면
웹사이트가 어느 정도 다 만들어진
거거든요. 잘 만들어졌는지 테스트를
하기 위해서는 해당 폴더를 가서
cmd 명령어를 쳐 주시고요. 여기서

그리고 만들어진 로컬 서버를 실행을

만들어졌습니다. 근데 말씀드렸다시피
조금 빈약한게 보일 텐데 이미지들이
제대로 생성이 안 돼서 사이트가
빈약해 보입니다. 물론 이렇게 칼라를
지정한다거나 혹은 용량을 바꾼다거나
이런 건 제대로 작동을 하고
있거든요. 카트를 넣었을 때 카트에도
제대로 들어가는 모습이고요. 그래서이

있느냐? 안티 그래비티에 오셔서 현재

이미지들이 뭔지 리스트로 알려 달라고
요청하는 겁니다. 어디에 필요하고
어떤 사이즈로 필요하고 그리고 각각

가능한지 리스트를 알려 달라고 입력을
합니다. 자 그러면 재미나이가 각각
페이지들을 살펴보면서 어떤 이미지들이
필요한지 분석을 시작하거든요. 그리고
분석이 끝남과 동시에 어떤 이미지가
필요하고 어떤 프롬프트를 써야 되는지
이거를 리스트로 쭉 알려 주기
시작합니다. 자, 그러면 여러분은
이거를 그대로 쭉 복사를 하신
다음에요. 가셔서 도구에서 이미지
생성하기를 활성화를 해 주시고 방금
복사하신이 모든 프롬프트들을 여기다가
입력을 해 주시면 됩니다. 자,
그리고 엔터를 눌러 주시면요.이 모든
설정에 맞게 올바른 이미지들을 만들어
줄 겁니다. 자, 그 결과 여러분이

디자인의 이미지들이 완성이 되는 걸
확인할 수가 있습니다. 이거를 그대로

이미지가 다 완성이 됐으면 요거를

되는데요. 한 번에 이미지가 다섯
개밖에 옮겨지지가 않거든요. 그래서
다섯 개를 옮기고 이미지를 더
추가하겠다라고 입력을 하면 됩니다.
그리고 후발 주자들도 업로드를 대기를
해 주시고요. 뭐 딱히 이미지들에
대한 설명을 하지 않더라도 이미
여기에 어떤 프롬프트로 만들지를 적어
줬기 때문에 알아서 식별을 할
겁니다. 자, 그리고 나서는 알아서
업데이트를 해 줄 거거든요. 참고로
이미지를 업로드하실 때 여러분이
일일이 명령을 입력을 해 주셔야 될
때가 있습니다. 자, 다 완료가
되었고요. 웹사이트를 가서 확인을 해
보시면이 이미지들이 성공적으로 들어간
걸 확인할 수가 있습니다. 훨씬 더
이미지가 있으니까 더 깔끔하게
변했죠. 색깔 체인지도 자동으로
이미지가 바뀌는 걸 확인할 수가
있고요. 자, 근데 카트를 넣었을 때
아직 카트는 이미지가 제대로
표시되지는 않네요. 여러분이 바이브

이런 디테일에서 버그가 있다거나
오류가 나는 부분이 있습니다. 근데

힘들거든요. 그래서 필요한 또 다른
MCP 서버가 있습니다. 제가 아까
바이브 코딩은 세 가지라고 했죠.
컨텍스트도 필요하고 로직도 필요하고
디버깅도 필요합니다. 그래서 가장
중요한이 디버깅에 대해서 알아볼
건데요. 바로 테스트 스프라이트라고
하는 MCP 서버인데요. 지금 여기

코드에 디버깅까지 해 주는 미친

오른쪽 위에 MCP 서버를 무료로

셋업이 끝나고 들어오시면 무료
어카운트로 여러분이 크레딧개가 있는

충분하거든요. 그리고 더 좋은 건

현재 여러분이 해당 MCP 서버를
무료로 한 달 동안 사용할 수 있는
프로모션이 있습니다. 그래서 현재는
제가 150 크레디밖에 없지만
왼쪽에서 무료 신청하기를 누른 다음에

버튼이 있죠? 해당 버튼을 클릭을
하시고 여기서 여러분이 무료 체험을
진행을 하시면 이렇게 150 크레딧이
아니라 550 크레딧으로 늘어난 걸
확인할 수가 있습니다. 즉 여러분이
훨씬 더 많은 프로젝트를 해당 MCP

같은 경우는 한 달 뒤에는 19불씩
차지가 되기 때문에 한 달을 무료로
사용을 해 보시고 결정을 하시면

그러면 여러분이 API 키를 만들

만들어 주시고요. 테스트
스프라이트라고 이름을 넣고
만들겠습니다. 자, 그리고 역시나
또다시 API 키가 발급이 되는데이
키도 복사를 해 두셔야 됩니다. 자,
그리고 아까 셋업으로 돌아가서 저희는
커서나 클러드 코드를 사용하는게
아니기 때문에 저희는 안티 그래비티를
사용하죠. 가장 마지막 탭을 선택을
해 주시고 여기 나와 있는 코드를
복사를 해 줍니다. 안티 그래비티에
가면 똑같죠. 우측에서 옵션 눌러
주시고 MCP 서버 클릭한 다음에
매니지 MCP 서버 여기서는 컨피그
파일 들어가 주시고 저희가 복사한 걸
여기에 붙여 넣기를 합니다. 그리고
저희가 다시 기존의 MCP
서버들에서를 입력하고 엔터를 눌러
주시면 안티 그래비티가 자동으로 내가

지우도록 하겠습니다. 그리고 여기에
API 키를 넣어야 되잖아요. 아까
여러분이 만드신 키를 복사한 다음에이
값에다가 붙여 넣기를 해 주시면
되겠습니다. 이렇게 테스트
스프라이트까지 MCP 서버 설정을
완료를 한 다음에 다시 한번 새로
고침을 해 줍니다. 자, 근데 에러가
날 수가 있거든요. 이거는 아직 안티
그래비티에서 서버 설정을 조금 더
업그레이드를 해 줘야 되는 부분이
있는데 이거를 고치는 방법이
있습니다. MCP를 돌아와서요. 방금

하단에 한 줄을 더 추가를 해 주시면
