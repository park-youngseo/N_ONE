```
네, 안녕하세요. 다들 안티그래피티
잘 쓰고 계신가요? 어, 저는
안티그래피티를 사용해서 다양한
웹서비스를 만들고 있는데요.
안티그래피티를 사용하면서 느낀네 가지
팁을 한번 여러분들께 공유해 드리도록
하겠습니다.
네, 우선 여러분들 안티그래프티
안에서 나노바나가 지원되는 거 알고
계신가요? 그래서 저 같은 경우에는
나노만 하나를 이용해서 웹페이지를
기획합니다. 그래서 예시로 간단하게
설명해 드리겠습니다.
물건을 판매하는 랜딩 페이지 예시
이미지를 생성해 줘라고 입력을 하게
되면 얘가 나노바나를 통해서 예시
이미지를 생성해 줍니다. 그러면이
제미나 3프로 플래닝 이것을 설정한
것을 확인하시고 엔터를 눌러
주겠습니다.
네. 그러면 이런 식으로 이미지가
생성이 되는데이 이미지를 오픈을 해
보겠습니다.
이미지를 오픈했을 때 이런 식으로
나오게 됩니다. 근데 내가 여기서
수정하고 싶은 부분이 있다라고 하게
되면 이렇게 드래그를 하시고
그다음에이
글자를 한번 저는 바꿔 보도록 할
텐데요. 그래서 이런 식으로 해당
글자를 입체감 있는 사운드로 바꿔
주세요. 그리고 어디 커멘트 여기
리뷰에 하나가 들어가 있다라는게
뜹니다. 그러면 리뷰를 눌러 주시고
리뷰 원커멘트를 눌러 주시면 여기
어떤 식으로 내가 리뷰를 했는지가
나옵니다. 그러면 아래의 요청에
따라서 글자를 변경해 주세요라고
입력해 보겠습니다.
네. 그러면 수정이 완료가 되어서
이런 식으로 나오게 되는데 눌러
주시면 입체감 있는 사운드라고 반영된
것을 알 수 있습니다. 그래서
여러분들께 드릴 팁은 내가 만약에
기획을 하고 나서 그 이미지를
수정하고 싶을 때 그냥 드래그 하시고
거기다 커멘트를 작성해 주기만 한다면
이게 내가 원하는 형태로 변경할 수
있다라는게 팁입니다. 그래서 이렇게
나노바나를 이용해서 기획이 다
되었다면 이걸 기반으로 웹을 만들어
달라라고 하게 되면 내가 그 이미지
그대로 웹을 만들 수 있습니다. 한번
요청해 보도록 하겠습니다.
네. 저는 사실 뭔가 만들기 전에
이런 식으로 이미지를 만들어 놓은
다음에 이걸 기반으로 만들어 달라고
요청을 하는데요. 이것도
마찬가지입니다. 저 같은 경우는 위에
이미지를 참고해서 맵을 만들어 줘라고
입력을 해서 어 얘가 개발을 해
줍니다. 개발해 주고 베리파잉까지 해
줍니다. 그러면 랜딩 페이지 구형이
완료가 되었다고 나오네요. 한번
확인해 보도록 하겠습니다. 네. 이런
식으로 아까 이미지를 만들어 냈을
때랑 되게 유사한 랜딩 페이지가
만들어진 것을 알 수 있습니다.
그래서 여러분들도 뭔가 웹을 바로
만들기보다는 이렇게 이미지로 먼저
랜딩 페이지 이미지를 만든 다음에 그

하면 더 잘해 줍니다. 여러분들

수 있습니다. API 키를 설정해야
되는 것도 있지만 몇몇 개는 설정할

사용해 보시는 거 권장드리겠습니다.

서버들이 있습니다. 뭐 퍼플레이시티

원하는 MCP 서버가 없다라고 하게
된다면이 매니지 MCP 서버를 눌러
줍니다. 그다음에 여기서 뷰로우
컨픽에 들어갑니다. 그래서 여기서이
제이슨 형태로 넣어 주는 것이 있는데
여기서 내가 원하는 MCP 서버를
구성 요소를 여기다 넣어 주시면
됩니다. 그러면 여기서 넣어 준
다음에 컨트롤 S를 누르시고
저장하시게 되면 여기 매니저 MCP에
여기 뜨게 됩니다. 저 같은
경우에는이 파일 시스템 MCP 서버와
플레이라이트 MCP 서버를 이런 뷰로
컨피 안에다가 넣어 줬습니다.
그랬더니 잘 등록이 되었습니다. 그
등록이 잘 되어졌다면 이런 식으로 어
무슨 도구들을 사용할 수 있는지

보겠습니다. 네. 그러면 이런 창이

진행하였고
그다음에 다시 저기가 지우고 다시
네, 곱하기를 합니다.

이제는 나누기까지 수행하는 것을 볼
```
