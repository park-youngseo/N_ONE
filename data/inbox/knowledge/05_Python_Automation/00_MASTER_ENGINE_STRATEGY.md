# [Master 05-01] 파이썬 무인 자동화 엔진: 에이전트 및 견고화 전략

## 1. AI 에이전트 연동 (Agentic Python)
- **핵심**: 파이썬을 AI의 '손과 발'로 활용하는 기술.
- **실전**: 
    - `OpenAI API` / `Anthropic API`를 파이썬 코드에 심어, AI가 크롤링한 데이터를 분석하고 그 결과에 따라 다음 동작을 스스로 결정하게 함.
    - 예: "네이버 데이터랩에서 1위 키워드를 찾아내고(Scraping), 그에 맞는 블로그 글을 써서(AI), 자동으로 포스팅해라(Automation)."

## 2. 불멸의 시스템 (Robustness & Error Handling)
- **핵심**: 오류가 발생해도 시스템이 멈추지 않고 수익을 계속 창출하게 함.
- **기술**: 
    - **Try-Except-Retry**: API 호출 실패나 네트워크 오류 시 5분 뒤에 자동으로 재시도하는 로직.
    - **Logging**: 모든 동작을 `logs/` 폴더에 기록하여, 사령관님이 자고 일어났을 때 밤새 무슨 일이 있었는지 한눈에 보고받게 함.

## 3. 완전 무인 가동 (Scheduling)
- **핵심**: 인간의 개입을 0%로 만드는 스케줄링.
- **방법**: 
    - **Windows Task Scheduler**: 파이썬 스크립트를 윈도우 서비스로 등록하여 매일 새벽 3시에 자동 가동.
    - **Telegram Alert**: 작업이 완료되거나 치명적 오류 발생 시, 사령관님의 텔레그램으로 즉시 상황 보고.

- **01번 시나리오 엔진**: 파이썬이 자동으로 트렌드 키워드를 수집해 시나리오 소재를 던져주고, AI가 쓴 시나리오를 PDF로 변환해 사령관님의 메일로 전송.
