# [Master 17-01] AI 콘텐츠 자동화: Claude Code & Gemini 실전 워크플로우

## 1. 개요
- **출처**: CocojuanTV (AI 생산성 전략 분석)
- **핵심 가치**: 1인 제작자가 AI를 '팀원'으로 활용하여 대량의 고퀄리티 콘텐츠를 생산하는 시스템.

---

## 2. Claude Code 기반 영상 제작 자동화 로직

### [Phase 1: 데이터 학습 및 기획]
- **로직**: 타겟 채널의 성공한 영상 스크립트 5개를 Claude에게 학습시킨다.
- **결과**: 해당 채널의 '터지는 말투', '자막 배치 타이밍', '정보 전달 순서'를 완벽히 복제한 기획안 도출.

### [Phase 2: 원터치 콘텐츠 생성]
- **Gemini & NotebookLM 활용**: 방대한 전문 지식을 NotebookLM에 넣고, Gemini를 통해 블로그 포스팅이나 유튜브 대본으로 즉시 변환.
- **Claude Code 실행**: 변환된 대본을 기반으로 영상 편집에 필요한 자막 파일(SRT)과 나레이션 가이드를 자동 생성.

---

## 3. SEO 및 유입 극대화 전략

### [검색 엔진 최적화 (SEO)]
- AI가 생성한 대본 내에 **[연관 검색어]**와 **[LSI 키워드]**를 자연스럽게 배치하는 프롬프트 기술 적용.
- 유튜브 썸네일과 제목에 클릭을 유도하는 '심리적 트리거'를 AI가 3가지 옵션으로 제안하게 함.

---

## 4. NONE-OS 시스템과의 시너기
- 사령관님의 **[AI 사운드 엔진]**은 이 자동화 파이프라인의 **'마지막 퍼즐'**입니다. 
- AI가 만든 영상에 사운드 엔진으로 **'스튜디오급 현장감'**을 더하면, 시청자는 이것이 AI가 만든 영상인지 실제 사람이 만든 영상인지 구분하지 못하게 됩니다. (압도적 품질 경쟁력 확보)
