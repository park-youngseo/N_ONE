# 주요 작법 거장 알고리즘 (Core Screenwriting Harness)

## 1. Tyler Mowery_Harness
*   **[Golden Rule]**: 캐릭터의 외적 목표(Want)보다 내적 신념(Belief)과 진실(Truth)의 충돌이 서사의 중심이어야 한다.
*   **[Trigger Words]**: 
    *   "결심하다" -> 캐릭터의 Lie와 Truth가 충돌하는 씬 삽입.
    *   "대립하다" -> 단순한 싸움이 아닌, 가치관의 충돌로 묘사.
*   **[Anti-Pattern]**: 캐릭터가 고통 없이 성장하게 하지 말 것. 신념이 깨지는 고통이 필수.

## 2. John Truby_Harness
*   **[Golden Rule]**: 이야기는 유기적이어야 하며, 7단계 핵심 구조(Need~New Equilibrium)가 플롯을 관통해야 한다.
*   **[Trigger Words]**: 
    *   "계획" -> 주인공의 계획이 경쟁자(Opponent)와 동일한 목표를 겨냥하도록 설정.
    *   "전투" -> 물리적 전투를 넘어 도덕적 결단(Self-Revelation)이 동반되어야 함.
*   **[Anti-Pattern]**: 경쟁자가 주인공보다 약하게 설정하지 말 것. 경쟁자는 주인공의 거울이어야 함.

## 3. Michael Arndt_Harness
*   **[Golden Rule]**: 엔딩은 예측 불가능하면서도 논리적으로 완벽한 'Insanely Great' 상태여야 한다.
*   **[Trigger Words]**: 
    *   "포기하다" -> 물리적 구원 이전에 정서적 결단이 먼저 일어나는 씬 구성.
    *   "위기" -> 외부적, 내부적, 철학적 위기(Stakes)를 동시에 고조시킴.
*   **[Anti-Pattern]**: 데우스 엑스 마키나(갑작스러운 구원) 금지. 보상은 캐릭터의 희생 후에 온다.
