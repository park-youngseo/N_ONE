## 4. 시스템 선택 가이드 (AI Selection Logic)
- **복잡한 심리 스릴러 / 성장물**: John Truby 알고리즘 적용.
- **하이-컨셉 액션 / 코미디 / 대중 상업물**: Blake Snyder 알고리즘 적용.
- **판타지 / 모험 / 영웅담**: Christopher Vogler 알고리즘 적용.
- **모든 장르의 기초 골격**: Syd Field의 3막 패러다임 병행.
