# 캐릭터, 대사, 그리고 지문 (Dialogue & Parentheticals)

캐릭터의 목소리를 지면에 옮기는 기술적 방법입니다.

## 1. 캐릭터 이름 (Character Cues)
*   대사 바로 위에 대문자로 작성하며, 중앙 정렬에 가깝게 배치합니다. (3.7" 여백)
*   **Extensions**: 목소리만 들릴 때 `(V.O.)` - Voice Over, 화면 밖에서 들릴 때 `(O.S.)` - Off Screen을 이름 옆에 적습니다.

## 2. 지문 (Parentheticals)
*   대사의 톤이나 행동을 짧게 지시할 때 사용합니다.
*   **주의**: 남발하지 마세요. 배우의 연기를 방해하지 않을 정도로 최소한만 사용합니다.

## 3. 작성 예시 (Dialogue Example)
```text
                        SARAH (V.O.)
                I knew he would come back.

                        JACK
                (whispering)
                I never left, Sarah.
```

## 핵심 요약 (StudioBinder Basics)
| 요소 | 예시 및 용도 |
| :--- | :--- |
| **(V.O.)** | 내레이션이나 캐릭터의 속마음을 전달할 때 |
| **Parentheticals** | 대사의 의미가 오해받을 수 있을 때만 사용 |
