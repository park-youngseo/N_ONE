## 🔥 **B1단계｜다음 질문(Q123) – 포함단어(키워드) 저장 방 식** 

CSV의 마지막 컬럼이 “포함단어”인데, 

이건 형이 지정한 니즈 관련 키워드(예: 오디션, 프로필, 연기영상 등) 중 본문/댓글에 포함된 단어 목록을 저장하는 칸이야. 

예시로 본문이: 

코드 복사 

```
오늘처음으로오디션지원했어요! 프로필도새로찍었습니다.
```

이면 포함단어는: 

코드 복사 

```

```

이제 저장 방식을 선택해야 해.

## ✔ **장점**
