## 방법 ① **정적 페이지 요청 방식** 

`HTTP GET` 블록으로 HTML을 가져온 뒤 

HTML 안에서 **특정 부분만 뽑기(파싱)** 

   - 예: `<div class="title">` 에 있는 글제목들 

- AI 그래비티에서는 

   - “웹페이지 가져오기” → “HTML 파싱/Selector 설정” 같은 블록 조합. 

방법 ② **동적 페이지(스크롤, 버튼 클릭 필요)** 

- React/SPA 사이트처럼, 들어가자마자 빈 화면인데 

- 스크립트가 돌면서 나중에 내용이 뜨는 경우. 

- 이런 건 **헤드리스 브라우저** 개념이 필요해 

- (사람이 브라우저 켜서 클릭하는 걸 에이전트가 대신 하는 느낌) AI 그래비티에서 지원한다면: 

   - “페이지 열기 → 로그인 → 버튼 클릭 → 스크롤 → 현재 화면 HTML 가져오 기” 

- 만약 플랫폼에 이 기능이 약하면, 

   - 외부에 파이썬 / 셀레니움 스크립트를 둔 뒤 

   - AI 그래비티가 그 스크립트를 호출하는 구조로도 설계 가능.
