## [원본 내용 — 무손실 전문 (Part 2)]

> **[AX 하네스 준수]** 아래는 원본 자막 텍스트의 무손실 전문입니다.
> 요약 또는 압축 없이 원자 단위로 전체 내용을 보존합니다.

```
아니면 계획과 컨트롤 하에 하는 AI
증강 코딩의 모습이 되느냐로 나뉘게
됩니다. 자, 일단 여기서 우리가 좀
신경 써 줘야 되는 부분은이 각각의
단계들을 수행할 때 구체적으로 어떤
방법으로 해야 할지에 대한 그
아이디어를 제공해 줘야 된다는
부분입니다. 우리는 앞서서 정보를
조사하고 대본을 생성하는 등의 작업을
AI를 활용해서 일을 수행했습니다.
자, 잠깐이 AI의 특징에 대해서
조금 이야기를 해 보겠습니다. 우리가
AI를 정의할 때, 특히 용도에
대해서 이야기를 할 때 AI는 대본을
만들고 정보를 조사하는 용도로
사용하는 도구이다라고 콕의해서
이야기하진 않습니다. AI는
범용적으로 사용할 수 있는 그래서 그
용도는 사람의 창의성에 따라서
결정되는 도구입니다. 용도가 정해지지
않은 그래서 한계가 정해지지 않은
도구라는 점이 제가 AI에게 매력을
느끼는 이유이기도 합니다. 자,
근데이 창의성 그리고이 제치는 아직은
AI가 사람에게는 좀 뒤쳐진다고
생각합니다. AI를 번역기로 활용을
할 수 있겠으나 AI 스스로는 자신이
번역기로서 활용될 수 있다는 사실을
사람이 일겨워 주지 않는 이상
스스로는 자각하지 못하는 모습을 자주
보입니다. 에서 번역기를 만들어
달라고 하면 AI를 활용해서 번역을
하는 방식보다는 구글 트랜슬레이션
API와 같은 기존에 널리 알려진
번역 도구를 사용하는 모습으로
접근하려고 하는 모습을 자주
보입니다. 지금 이런 부분은 현재
AI의 활용 분야가 창의적이고
실험적인 제치가 필요한 경우가 더
많은 시점인데요. 시간이 흘러서 좀
더 상식적이고 일반적이게 된 AI
활용처들이 늘어나게 됐을 때 그때
AI도 우선적으로 그런 용도에 대해서
떠올릴 수 있게 될 거라고
생각합니다. 자, 그래서 지금 우리의
작업에서는 거의 대부분의 작업을
AI를 활용해서 하도록 되어
있습니다. 자, 그래서이 부분에서
AI를 사용하라고 지침을 해 줄
필요가 있습니다. 자, 그래서이
부분에 대해서 안티그래비티가 각각의
기능을 구현할 때 AI를 기반으로
해서 작동되는 코드를 만들도록 유도를
하기 위해서 사용할 제미나이 API
모델 이름을 명시를 해 줍니다.
대본과 이미지 생성 프롬프트의 생성도
마찬가지고요. 화면 이미지를
만들어내는 것도 마찬가지죠.이 이
화면 이미지 생성은 나노바나를
사용하도록 이렇게 나노바나 모델
이름을 지침을 해 줍니다. 자,
오디오의 경우는 오픈 AI의 API를
사용했습니다. 그래서이 API 모델
이름도 정확하게 작성을 해 줍니다.
자, 이렇게 해서 완성된이 내용을
그대로 복사해서 안티그래비티에게 보여
줍니다. 자, 이렇게 해서 이제
요청을 할 수도 있겠지만 이제 여기
조금 더 세부적인 추가 지침을 보태
줄 수가 있습니다. 자, 일단이
작동은 파이썬 언어로 코딩을 해
달라고 내용을 담아 주고요. 자,
그리고 AI가 자주 실수하는게
있는데요. 특정 기술에 대한 사용
방법에 대한 부분입니다. 지금은 이제
재미나이와 오픈 AI의 API를
사용하는 코드를 안티그래비티가 작성을
해야 되는 상황인데요. 이거에 실수를
하지 않을 수 있도록이 코딩 작업
전에 먼저 사용 방법을 조사를 하고
진행하라고 지침을 주겠습니다.
기본적으로 클로드 코드나
안티그래비티와 같은 코딩 에이전트들은
인터넷 검색 기능을 가지고 있습니다.
그래서 기본적으로 조사를 할 때이
검색 기능을 통해서 조사를 하게
되는데요. 자, 만약에 컨텍스트 같은
최신 기술 문서를 참고할 수 있도록
해주는 MCP를 연결해서 사용한다면
안티 그래비티가 웹 검색과 더불어서
이런 최신 기술 문서들도 직접적으로
접근을 할 수 있게 돼서 더 정확하게
작업을 해낼 수가 있습니다. 자,
그리고 지금 만들게 될 프로그램은
상당 부분의 기능들이 AI를 기반으로
작동합니다. AI를 기반으로
작동한다는 것은이 작업에서 단계마다
작동하는 그 AI가 주는 그 응답을
가지고 다음 작업을 이어간다는
것인데요. 다음 작업을 이어가기
위해서는 AI의 응답의 모습은 이처럼
비정형적인 모습이어서는 곤란합니다.
대신 이렇게 정형적인 모습이어야
하죠. 자, 그래서 추가 지침으로
AI를 사용할 때는 응답의 모습을
이처럼 정형적인 모습으로 응답을 받을
수 있도록 해 달라고 이야기를 해
줘야 됩니다. 자, 그리고이어서 이제
만들게 되는 프로그램은 GPT나
재미나의 의존에서이를 수행할 건데요.
자,이 의존을 하기 위해서 그 AI
기능을 사용하기 위한 권한을 담은
1년 번호를 함께 담아 줘야 됩니다.
이것을 API 키라고 부릅니다.
그래서이 API 키를 발급받아서 함께
제공을 해 줍니다.이 이 API 키는
앞서 설명드린 것처럼 그 사용을 위한
권한 마치 콘서트장이나 클럽에 가면
손목에 팔찌 같은 거 차죠. 그래서
그 공간에 있는 동안에는 손목에 차고
있어야 이제 입장이 가능한 건데요.이
손목 팔지와 같은 역할을 하는이
API 키를 이제 안티 그래피티가
만들게 될 그 프로그램 소스 코드
안에다가 함께 담아 달라고 요청을
하겠습니다. 자, 그렇게 하는 것을
코드에 하드코딩한다라고 부릅니다.
자, 이렇게 손목 팔지를 코드상에
하드코딩하는 것은 권장되는 방법은
아닌데요. 일단 여기서는 편의상
이렇게 하드코딩 하도록 하겠습니다.
자, 그리고 이제이어서 지금 만들게
될 프로그램은 이제 여러 가지
작업들을 할 텐데요.이 이 작업들을
할 코드를 모두 안티그래비티가 만들어
낼 수도 있겠지만 이미 그 작업을
하는 코드가 누군가에 의해서 잘
만들어져서 기터브 같은데 오픈 소스로
공유가 되어 있을 수도 있습니다.
자, 이렇다면 직접 만들 것이 아니라
그냥 그것들을 받아다가 쓰면 되겠죠.
그래서 그것을 받을 바구니에 해당되는
공간을 먼저 만들고이 안에서 작업을
진행해 달라는 요청을 이처럼 파이썬
가상 환경 만들고 작업한다라고 지침할
수 있습니다. 자, 그래서 이렇게까지
추가 지침들을 담아주고 요청을
합니다. 자, 그래서 요청한 대로
AI 사용 방법을 조사를 하고 작업
공간을 만들고 필요한 오픈소스
프로그램들 다운로드 받고 코드까지를
만들어 냅니다. 자, 이렇게까지 해서
이제 다 만들었으니까 이제 실행을 해
보겠냐고 이렇게 실행 명령어를 완성해
주면서이 실행을 제한을 합니다. 지금
제 경우는 제 안티그래비티 세팅은
이렇게 사람이 실행에 대해서 모두
검토하고 거부할지 수락할지를 결정하는
모습으로 되어 있는데요. 많은 바이브
코더분들께서이
부분에 대해서 그냥 모두
안티그래비티가 알아서 하도록 설정을
하고 사용을 하십니다. 물론
편리하지만 그럴 경우 가끔 AI가 할
수도 있는 오작동에 대해서는 무방비
상태가 된다는 단점이 있습니다. 그
오작동이 경우에 따라서는 치명적일
수도 있는데요. 얼마 전에 인터넷상에
안티그래비티를 사용하다가 컴퓨터상의
자료를 날렸다라는 경험에 대한
이야기가 공유되기도 했습니다. 말
그대로 중력의 영향을 받지 않고
데이터가 날아가 버린 사례입니다.
근데 사실 이게 또 이렇게 매번
확인을 해야 되고 한다면 번거로운게
사실이기도 한데요. 자, 안전을
챙기면서 편의도 가져갈 수 있는 방법
코드 깎는 노인 클래스에서 함께 알려
드리고 있으니까 많은 관심
부탁드립니다. 자, 어쨌든 이렇게
해서 완성된 프로그램을 수락해서
실행해 보겠습니다. 자, 실행 결과
지금 내용을 조사하고 대본을
만들고요.이어서 이미지 생성을 해
나가고 있습니다. 그리고이어서 오디오
생성을 하고 마지막으로 대본을 담는
텍스트 파일까지 만들어내는 것으로
해서 프로그램 실행이 완료가
됐습니다. 자, 확인해 보죠. 지금
이미지 파일 자, 꽤 어울리게 잘
만들어졌고요. 대사를 담은 텍스트
파일도 원하는 모습대로
만들어졌습니다.
&gt;&gt; 비우면 쑤시는 무릎, 지긋지긋한
관절염. 오디오 파일도 잘
만들어졌네요. 자, 이렇게 해서
프로그램 실행 한 번으로 앞서서 했던
그 반복적인 과정이 싹 사라질 수
있게 됐습니다. 자, 이렇게 해서
자료가 준비가 됐다면이 자료들을
가지고 영상으로 구원내는
프로그램을이어서 실행해 주는 것만으로
&gt;&gt; 비우면 쑤시는 무릎 지긋지긋
&gt;&gt; 동영상이 완전하게 만들어지는 것이죠.
자, 지금이 안티그래비티로 만든이
자동화 프로그램도 코드깎는 노인
클래스에서 함께 제공의
경우는이 쇼츠 영상을 만들어 보는
것에 대해서 이렇게 자동화를 해 본
것인데요. 이런 자동화는 어떤
분야에서든지 다양한 형태로 적용이
가능합니다. 이번 영상도 도움이
되셨기를 바라면서 또 다음 영상에서
인사드리겠습니다. 코드 깎는
노인이었습니다. 감사합니다.
```
