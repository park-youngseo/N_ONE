# 📌 NONE-OS: 에이전트 마스터 아키텍처 (2026-05-04 확정)

사령관의 모든 명령은 다음 **NONE-OS** 원칙에 따라 처리된다.

1. **에이전트 = 모델 + 하네스**: 지능(모델)보다 시스템(하네스)의 안정성과 규칙 준수를 우선한다.
2. **상태 유지(Stateful) 워크플로우**: 모든 작업은 `LangGraph` 방식의 그래프 구조로 설계되어, 중간에 끊기더라도 상태를 보존하고 복구한다.
3. **로컬 장악(MCP/A2A)**: 모든 행동은 로컬 파일 시스템, 도구, 에이전트 간 네트워크를 통해 직접 실행된다.
4. **무손실 지식 기반(RAG)**: AI의 추측보다 `knowledge/` 폴더의 무손실 데이터를 최우선 근거로 삼는다.
5. **자기 검증 루프(Self-Correction)**: 결과물 출력 전 반드시 '검토 에이전트'가 하네스 규칙 준수 여부를 확인한다.

4. **원문 인용 추출** — 핵심 발언 원문 그대로 따옴표 인용
5. **실행 과정 원자화** — 모든 절차를 번호 붙여 전수 나열
6. **AX 표준 파일 생성** — 300줄 이하 `.md` 저장
7. **원본 파일 유지** — 원본 `raw_data/` 파일 절대 삭제 금지
