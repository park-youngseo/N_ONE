# N-ONE 최신 AI 기술 리서치 (2025-2026)

## 1. GraphRAG: 지능형 관계형 검색
### 개요
단순 키워드 매칭을 넘어 지식 간의 연결망(Entity-Relationship)을 구축하는 RAG의 진화 형태.
### 핵심 이점
- **전체 맥락 파악**: 수백 개의 문서에 흩어진 정보를 하나의 테마로 엮는 능력이 탁월함.
- **추론 능력 향상**: "A가 B에게 영향을 준 이유" 같은 복합적 질문에 대한 정답률 상승.
### 하네스 적용 방안
- `knowledge/` 내의 모든 파일을 그래프 DB(또는 마인드맵 형태)로 연결하여 시나리오 세계관의 일관성 유지.

## 2. SLM (Small Language Models) & 로컬 런타임
### 개요
7B~14B 규모의 작지만 강력한 모델들이 클라우드 모델(GPT-4)의 성능을 위협함.
### 대표 모델
- **Qwen 2.5 (14B/32B)**: 코딩 및 수학적 추론에서 독보적 성능.
- **Llama 3.2**: 모바일 및 로컬 환경 최적화.
### 하네스 적용 방안
- 모든 '노가다성' 작업(수집, 요약, 분류)은 100% 로컬 SLM으로 처리하여 API 비용 Zero화 달성.

## 3. Agentic Design Patterns (앤드류 응)
### 4대 핵심 패턴
1. **Reflection (반성)**: 결과물을 내기 전 스스로 비판하고 수정함.
2. **Tool Use (도구 사용)**: 필요한 정보를 위해 외부 API나 코드를 실행함.
3. **Planning (계획)**: 목표를 위해 동적으로 단계를 나누고 조절함.
4. **Multi-agent Collaboration (협업)**: 서로 다른 전공의 에이전트들이 토론하여 최적안 도출.
### 하네스 적용 방안
- 현재 하네스의 `orchestra` 전략을 이 4대 패턴에 맞춰 고도화.

## 4. 로컬 멀티모달 (Local Multi-modal)
### 개요
이미지, 음성, 영상을 외부 클라우드 없이 로컬(Ollama/Whisper)에서 즉시 처리.
### 하네스 적용 방안
- 형님이 주시는 음성 메모나 레퍼런스 이미지를 하네스가 즉시 읽고 시나리오 설정으로 변환.
