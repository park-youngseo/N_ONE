유튜브 소재 발굴기를 만들어 보려고
합니다. 키워드를 넣으면 잘 터진
영상을 찾아주고 댓글을 분석해서
새로운 소재까지 추천해 주는
프로그램을 만들려고 합니다. 오늘도

따라하만 해 주시면 됩니다. 그럼
먼저 안티 그래비티가 뭔지 간단히
설명드리겠습니다. 구글이 만든 AI
코딩 도구인데요. 혹시 커서나 클로드
코드에 대해서 들어보셨을까요? 그거랑
조금 비슷한데 구글 버전이라고
생각하시면 됩니다. 근데 이게 조금
특이한게 있습니다. 코드만 짜 주는
것이 아니라 브라우저까지 직접 열어서
테스트를 진행해 줍니다. 내가 만든
프로그램이 잘 돌아가는지 AI가
알아서 확인해 주는 거죠. 현재
무료이고 맥, 윈도우, 리스 다
지원하고 있습니다. 설치는
간단합니다. 안티그래피t.
구ogle에 들어가셔서 다운로드 받고
설치하신 다음 구글에 로그인하면
끝입니다. 제가 그동안 클로드 코드를
활용해서 코딩하는 법을 알려 드리고
싶었지만 그 안에 진입정력이 있기
때문에 웹에서 하는 방법만 알려
드렸습니다. 때마침 안티그래비티가
출시되어서 잽싸게 알려 드리려고 이번
영상을 촬영하게 되었습니다. 근데
오늘은 안티그래비티만 쓰는게 아니라
구글 AI 스튜디오랑 같이 쓰려고
합니다. 이유는 안티그래비티가
무료이긴 한데 사용량이 제한이
있습니다. 그래서 프로그램을 만들다
보면 제한에 걸려서 기다려야 하는
경우가 발생하기 때문에 크롬프트 작성
같은 경우는 구글 AI 스튜디오에서
진행을 하려고 합니다. 그럼 안티
그래비티를 다운서 설치해 보겠습니다.
네. 설치한데 약 1분 정도가
소유되었습니다. 프로그램을 바로
실행해 보겠습니다.
아, 저는 이미 한번 로그인을 한
이력이 있기 때문에 바로 로그인이
됐는데 잠시 로그 아웃을 해
보겠습니다. 우측 상단에 보시면
로그인드 구글인데 눌러 주시고
계정을 선택해 주시면
됩니다.
그래피티를 열어 보겠습니다.
이제 차음 보는 프로그램 화면에서
적분 당황하실 수도 있는데 우선 오픈
폴더를 눌러서 프로그램을 만들 폴더를
한 개 만들어 주시면 됩니다. 저는
폴더명을 소재 추출기라고 해서 폴더를
만들어 보겠습니다.
여기서 예스를 눌러 주시면
안티그래비티가 해당 폴더에 모든
파일에 접근할 수 있도록 권한을
부여한다라고 생각해 주시면 됩니다.
사용할 수 있는 모델은 제미나입 프로
소네 4.5 그다음에 GPT 모델이
있습니다. 그리고 왼쪽에 보시면
플래닝과 패스트 모드가 있는데
플래닝은 실행하기 전에 조금 더
계획을 세우고 진행을 한다고
생각하시면 되고 패스트 같은 경우는
요청하는 발을 바로바로 실행하는
모드라고 생각하시면 됩니다. 일단
저희는 플래닝 모드로 진행을 할
예정입니다. 로그인 옆에
세팅 버튼을 눌러 주시면 에디터
세팅이 있습니다.
에이터 세팅에서 안티 그래비티
세팅으로 눌러 주시고
아티팩트에서 올이지 프로시들
바꿔 주시고 터미널에서 터로 바꿔
주시면
파일 엑세스도 오토로 바꿔 주시고
오토 컨티뉴까지 켜 주시면
웬만하면 확인받지 않고 계속 끝까지
진행을 할 겁니다.
그럼 오픈 AI로 이동을 하겠습니다.
자, 오픈 AI 스튜디오에 들어오시면
우측 상단에 선택된 모델이 제임이
3프리뷰인지 먼저 확인해 주시면
됩니다. 아니라면 눌러서 선택해
주시면 됩니다. 제일 좋은 모델을
무료로 사용할 수 있으니 사용하지
않을 이유가 없죠. 그러면 오늘 만들
프로그램에 대해서 먼저
설명드리겠습니다. 오늘은 유튜브 소재
발구기를 만들 건데 제가 생각하는
기능은 총네 가지입니다. 첫째,
키워드로 영상을 검색합니다.
예를 들어 바이브 코딩이라고 치면
관련 영상이 쭉 나오겠죠.
줄째 구독자 대비 조회수 비율을
계산해 주는데 이게 핵심이라고 볼 수
있죠. 구독자가 1만 명인데 조회수가
100만이 나오면이 영상이 알고리즘을
타고 있다는 거니까 그런 영상을 찾아
달라고 하는 겁니다. 셋째, 그렇게
필터링된 영상 중에서 특정 영상을
선택하고 그 영상에 댓글을
수집합니다.
넷째, AI가 댓글을 분석해서 새로운
소재를 추천해 주는 거죠. 그리고 그
소재를 선택하면 간단한 대분 목차까지
만들어 줍니다. 다시 정리해서
말하자면 키워드만 입력하면 소재를
추천받고 그 소재로 간단한 목차까지
받을 수 있는 프로그램을 만드는
거죠. 이거를 다 단일 HTML
파일로 만들 예정입니다. 아, 참고로
오늘은 API 키를 코드에 직접
집어넣는 하드코딩 방식으로 진행할
겁니다. 배포하실 예정이라면
하드코딩은 하시면 안 됩니다. 오늘
프로그램에는 유튜브 API키랑 제미나
API키가 필요한데 발급 방법은 지난
영상을 기준으로 해서 간단히 보여
드리겠습니다. 이미 발급받은
API키가 있으신 분들은 스킵하시면
됩니다. 먼저 구글 클라우드 컨솔에
접속해 주시면 됩니다. 검색하셔도
되고 더란의 링크를 클릭하셔서
접속하셔도 됩니다. PI키를 발급받기
위해서는 구글 계정이 필요하니 구글
계정으로 로그인해 주시기 바랍니다.
먼저 접속하시면 상단 메뉴에서
프로젝트 선택을 하셔서 새 프로젝트
만들기를 해 주세요. 프로젝트 이름은
아무거나 입력하셔도 됩니다. 가운데에
보시면 API 및 서비스 버튼이
있습니다. 아, 참고로 상단의
프로젝트는 행성하신 프로젝트가
선택되어 있어야 합니다.이 버튼을
눌려 주세요. 이렇게 화면이
전환되셨다면 좌측에 라이브러리를
선택해 주세요. 유튜브 데이터
API, V3를 입력후 검색하시면
결과값이 하나만 나오는데 결과를
클릭하시고 사용 버튼을 눌려 주세요.
사용 설정을 완료하셨다면 자측 상단에
구글 클라우드를 다시 한번 눌러
주시길 바랍니다. 그러면 다시 처음
화면으로 돌아오게 되면 여기서 API
및 서비스를 다시 눌러 주시기
바랍니다. 이제 자측에서 사용자 인증
정보를 눌러 주세요. 여기서 상단에
사용자 인칭 정보 만들기를 눌리시고
첫 번째는 API 키를 눌러 주시면
됩니다. 그러면 API 키를 만들 수
있는데 이름은 이번에도 편안한 대로
입력을 해 주시고 제한 사항을
무시하신 다음에 만들기 버튼을 눌러
주시면 됩니다. 이걸로 API 키
발급받는 건 끝이고이 API 키를
복사하셔서 메모장에 잘 기록해 두시기
바랍니다. 오늘 발급받을 재미나
API키는 무료로 사용할 수 있으며
1일 사용량 체한이 있지만 개인이
쓰기엔 충분합니다. 우선 구글에 구글
AI 스튜디오를 검색해 주세요. 구글
AI 스튜디오를 접속해 주시면 여기서
자측 하단에 있는 겟 API 키를
눌러 주시면 됩니다. 그리고 우측
상단에 API 키 만들기를 눌러
주세요. 키 이름은 아무거나 입력해
주시면 되고 프로젝트를 선택해 주시면
됩니다. 프로젝트 이름도 설정해
주겠습니다.
오른쪽에 카피 API 키 버튼을 눌러
주시면 API 키를 복사할 수
있으며이 API 키든 메모장에 잘
저장해 주시기 바랍니다. 혹시나 영상
중간에 제i 키가 노출될 수도 있는데
저는 영상 촬영 후 폐기할 예정이라
제 거 따라 쓰시지 마시고 꼭 직접
발급받아서 사용하시길 추천드립니다.
자, 이제 만들어 볼 건데요. 저희는
프롬포트를 먼저 만들 예정입니다. 역
프로 프롬포팅이란 방식을 활용할 건데
내가 원하는 걸 설명하면 AI가
프롬프트를 대신 작성해 주는 거죠.
저는 A에게 이렇게 전달할
예정입니다. 아까 프로그램을 만들
거라고 설명드렸던 내용을 불로
입력하는 겁니다. 나는 유튜브 소재
발굴기를 만들고 싶어. 키워드로
영상을 검색하고 구독자 대비 조회수
비율 계산해서 잘 터진 영상을 찾고
댓글을 수집해서 AI를 분석한 다음
새로운 소재를 추천받는 기능이야.
단은 HTML 파일로 만들 건데 이걸
만들기 위한 프롬프트를 작성해 줘.
유튜브 데이터 API랑 재미나
2.5플 API를 사용할 거고 API
키는 코드에 직접 하드코딩할 거야.
아래 API 문서를 참고해서 작성해
줘. 그대로 복사해서 입력해
그리고 참고하라는 API 문서는
유튜브에서 제공하는 공식 API
문서, 제미나에서 제공하는 공식
API 문서입니다. 링크는 다보기에
같이 올려 놓겠습니다. 이제 링크까지

우선 간단하게 내용을 확인해 보면
유튜브 데이터 API와 제미나

작성을 해뒀죠. 한 번 더 제작성해

그러면 이제이 프롬프트를
복사하겠습니다. 점점점을 눌리신 다음
