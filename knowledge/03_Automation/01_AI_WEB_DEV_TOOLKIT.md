# [Master 06-02] 실전 AI 웹 개발 도구 및 배포 (The Toolkit)

사령관님, 느낌(Vibe)을 현실로 바꾸기 위한 구체적인 무기들과 그 사용법입니다.

## 1. 최강의 무기: Cursor (AI Code Editor)
- **핵심 기능**: `Cmd + K`(한 줄 수정), `Cmd + L`(전체 코드 채팅).
- **Vibe 전략**: "파일 전체를 읽고, 지금 이 디자인을 애플 공식 홈페이지 같은 감성으로 다 뜯어고쳐 줘."
- **실전 팁**: `.cursorrules` 파일을 만들어 사령관님이 선호하는 코딩 스타일과 디자인 원칙을 AI에게 미리 주입하십시오.

## 2. UI 생성기: v0.dev (by Vercel)
- **사용법**: "넷플릭스 같은 대시보드 만들어줘"라고 입력하면 리액트(React) 코드가 즉시 생성됩니다.
- **연결**: 여기서 만든 UI 코드를 그대로 복사하여 Cursor에 붙여넣으면 개발의 80%가 끝납니다.

## 3. 완전 무인 배포 (Deployment)
- **Vercel / Netlify**: 사령관님의 코드를 드래그 앤 드롭하는 것만으로 `https://commander-app.vercel.app` 과 같은 고유 주소를 가진 웹사이트가 탄생합니다.
- **수익화**: 여기에 결제 모듈(Stripe 등)만 붙이면 즉시 전 세계를 상대로 장사를 시작할 수 있습니다.
