오랫동안 디자인으로서 15년 정도

하는 웹사이트인데 여기에 들어가

있고 이렇게 별도로 쫙 그릇을 선택을

바로 알 수 있는 그런 사이트다라고
할 수가 있는 것이죠. 이것을
재미나이 AI 스튜디오에 제가 한번
입혀 보고자 합니다. 한번 지금부터

하고 조건 같은 걸 걸어 줘야겠죠.

드래그 앤 드롭 왔다 갔다 할 수

안티그라피티로 한번가 볼게요. 네.
여러분 안티그라피티에 도착을

그럼 여기서 열기를 누른 다음에는 뭘
할 것이냐? 지난번 제가 다른
영상에서도 설명드렸다시피 여기서 해야
될 일은 환경 설정을 해 달라라고
말하는 것입니다. 환경 설정을 해
줘라고 이제 얘기를 하면은 제가 환경
설정을 마무리해서 저 얘기 알려 줄
거예요. 환경 설정을 해 줘야 되는
이유는 무엇이냐면 개발 도구에서는
워낙 다른 언어로 개발하시는
개발자들이 많기 때문에 굉장히
유연하게 설계가 돼야 됩니다. 그래서
환경이 무엇인지에 대해서 미리 세팅을
하나 해 놓고 가야 되는 그런 살짝
번거로움이 존재하기는 하죠. 그래서이

일은 뭐냐면 여기에 API 키를
집어넣는 것입니다.이 API 키는
어디서 우리가 다운을 받냐면 구글

메뉴에 존재하다 보니까 여기서부터
여러분들이 API 키 만들기를 누르고
새롭게 시작을 해 보면 되는 것이죠.
뭐 아까 우리가 프로젝트 이렇게
저장을 했으니까 이렇게 만들어 보고
저희 기존 프로젝트에 키 만들기를
누르겠습니다. API 키는 절대로
외부에 공개하시면 안 되고요.이
영상에서는 한번 쓰고 삭제하도록
하겠습니다. 그래서 요거 같은
경우에는 이렇게 API 키를 생성을

이렇게 재미나의 AP 키라고 하는
부분에 이렇게 붙여 넣기를 하면은
되는 것이죠. 이렇게 해서 붙여
넣기를 하게 되면은이 프로젝트를 켜게
됐을 때 곧바로 아까 등장하지 않았던
이미지들이 한꺼번에 나타나는 그런
경험을 할 수가 있을 것입니다. 자,
그렇게 되면은 이제 환경 설정이
완료됐다라고 이렇게 0으로 뜨게
되는데요. 여기서 우리가 해야 될
일은 뭐냐면 npm런 데브를 입력을
해 보는 것입니다. 이렇게 됐을 때는
npm 런데가 이렇게 뜨면서 지금
여러분들이이 로컬 호스트를 누르면은
미리 보기를 할 수 있는 그런 구조로
설정이 될 수 있다라는 것을
말씀드리고 싶어요. 그럼 이거 한번
눌러 볼게요. 네. 처음에 우리가
진입을 하게 되면은 이렇게 그릇들이
계속 이미지가 생성된 걸 보실 수가
있습니다. 이렇게 해서 한번 주마우을
해 볼까요? 주마우 해 보면은 네,
이렇게 이미지들이 하나씩 생성이 되고
있는 걸 볼 수가 있어요.
네, 이렇게 하나씩 하나씩 뿅뿅
나타나고 있죠. 그러니까 지금 현재이
프로토타입은 일단 이미지를 직접
재미나이가 다 생성하게끔 만드는 그런
조건을 가지고 있다라고 보시면

있죠. 이렇게 해서 전반적으로이

걸 만들어 줬어요. 그러니까이

설명 이런 것들을 생성해 주고
있다라는 걸 또 알아서 해 주고 있는

같아요. 그래서 모션 같은 경우가
이렇게 굉장히 디테일하고 여름으로 좀
재밌는 어 인터액션 한번 만들어 본

여러분도 만들 수 있다라는 것을
