# [Master 12] AI 시나리오 마스터 지식 엔진 v2.0 (Story Soul)

> **"이야기는 데이터가 아니라, 인간 심리의 간극(The Gap)에서 탄생한다."**

## 1. 하이브리드 작법 인프라 (Infrastructure)

### 1-1. Claude 4.7 / Gemma 4 로컬 구동
- **도구**: Ollama + VS Code (Continue 익스텐션 연동).
- **강점**: 사령관님의 비공개 시나리오(01번 폴더) 데이터를 외부 유출 없이 로컬에서 무제한 분석.
- **워크플로우**: `MASTER_SHEET.md`를 항상 컨텍스트의 최상단에 고정하여 '캐릭터 드리프트'를 원천 봉쇄함.

### 1-2. Sudowrite & FinalBit 연동
- **Sudowrite**: 창의적 '영감'이 필요할 때 '묘사 확장' 및 '대사 리믹스' 기능 활용.
- **FinalBit**: 실제 프로덕션 단계의 표준 포맷팅(Scene Heading, Action, Character, Dialogue) 준수.

---

## 2. 로버트 맥키 & 존 트루비 하네스 프로토콜

### 2-1. 맥키의 '간극(The Gap)' 강제 적용
- AI에게 장면 집필 전 다음의 논리 회로를 강제함:
    - **Step 1**: 캐릭터의 의도(Intention) 정의.
    - **Step 2**: 캐릭터가 예상하는 긍정적 결과 정의.
    - **Step 3**: 실제 벌어지는 예기치 못한 반전(The Gap) 설계.
    - **Step 4**: 간극에 의한 캐릭터의 감정적 타격(Reaction) 묘사.

### 2-2. 트루비의 22단계 구조 검증
- 전체 플롯을 존 트루비의 서사 구조로 분해하여, '도덕적 결함(Moral Flaw)'과 '자기 발견(Self-Revelation)'이 정점에 도달하는지 실시간 감시.

## 3. 실전 제작 브릿지 (Visual & Audio)

### 3-1. LTX Studio 시각화 콘티 생성
- 완성된 스크립트를 LTX Studio 전용 비주얼 프롬프트로 자동 변환.
- 샷 사이즈(Close-up, Wide), 카메라 무빙(Dolly, Pan) 정보를 대본에 메타데이터로 포함.

### 3-2. 음성 피드백 루프
- ElevenLabs 무제한 루프를 통해 조(Joe)와 니코(Niko)의 대사를 실제 육성으로 출력.
- "읽기 좋은 대본"이 아닌 "들었을 때 소름 돋는 대본"으로 최종 정제.

## 4. 사령관님을 위한 행동 지침 (SOP)
1. **[Structural First]**: 대사를 쓰기 전, AI에게 전체 장면의 '가치 변화(Value Shift)' 도표부터 제출하게 할 것.
2. **[Stress Test]**: 블록 작성이 끝나면 Claude 4.7에게 "이 장면을 완전히 망치는 법"을 묻고, 그 반대로 수정할 것.
3. **[Incognito No-Go]**: 로컬 LLM 사용 시에는 상관없으나, 클라우드 우회 사용 시에는 인증 차단을 막기 위해 일반 창 모드를 유지할 것.
