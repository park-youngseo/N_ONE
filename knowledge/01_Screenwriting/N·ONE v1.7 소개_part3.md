**1.** 지망생 업로드 → 얼굴 랜드마크(길이·폭·눈간격·코길이 등)·분위기 임베딩 사전계산 

**2.** 서버 분류/축적 → `age_band` 버킷·키 관리, 벡터 캐시 

**3.** 제작사 레퍼런스 업로드 → 동일 방식으로 랜드마크·임베딩 추출 

**4.** 매칭 → `Score = 0.5(기하) + 0.4(분위기) + 0.1(연령대)` 초깃값으로 상위 N 추천 + **설명문** 자동 생성 

   - 예: “세로/가로비 0.93 유사[사실], 눈간격/코길이 비율 차이 3% 이내[사실], 분위 기 임베딩 상위 10% 근접[추론].”
