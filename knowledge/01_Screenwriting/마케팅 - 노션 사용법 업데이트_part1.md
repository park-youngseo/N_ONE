## **1) 내가 실시간 업데이트를 못 하는 이유** 

내 지식은 **학습 데이터 시점** 으로 고정 

- 새 버전 UI가 나오면 

- → **형이 보고 있는 화면 = 최신 버전** 

- → **내가 알고 있는 화면 = 이전 버전** 

그래서 버튼 위치·메뉴 명칭·우측/좌측 이동 같은 변화는

## **2) 그럼 왜 스크린샷이 필요하냐**

- → 내가 즉시 **정확한 위치/버튼/구조** 로 설명 가능
