# auto manage — System Diagnostics and Management

> [!NOTE]
> Global Connectivity: [Registry](../SKILL.md) | [Verify](verify.md)

## auto doctor — Health Diagnostics

Check environment, dependencies, and configuration.
Report missing tools (gh, ast-grep, etc.).

## auto update — Update Harness

Update `.claude/*`, `.codex/*`, `.gemini/*` files from templates.
Preserve user customizations outside `AUTOPUS:BEGIN` markers.

## auto platform — Platform Management

List, add, or remove AI coding platforms.
Manage provider configurations in `autopus.yaml`.
